Hero Banners API Test

Test the hero banners API with different depth levels to see complete image data.

1. Get Hero Banners with Complete Image Data (Default)

This will fetch hero banners with depth=2, providing complete image information including all sizes and metadata.

GET /api/hero-banners

2. Get Hero Banners with Basic Image Data

This will fetch hero banners with depth=1, providing basic image information.

GET /api/hero-banners?depth=1

3. Get Hero Banners with Minimal Data

This will fetch hero banners with depth=0, providing only image IDs.

GET /api/hero-banners?depth=0

4. Get Featured Hero Banners

This will fetch only featured hero banners with complete image data.

GET /api/hero-banners?featured=true

5. Get Hero Banners in Vietnamese

This will fetch hero banners in Vietnamese locale with complete image data.

GET /api/hero-banners?locale=vi
